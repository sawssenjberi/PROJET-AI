#  DermAI Detect - Espace de Diagnostic Clinique

DermAI Detect est une application web médicale basée sur l'intelligence artificielle pour la classification et le diagnostic automatisé des lésions cutanées (Bénigne vs Maligne). 

##  Fonctionnalités
- **Authentification Sécurisée :** Espace de connexion pour le personnel clinique.
- *Tableau de Bord Centralisé :** Interface d'accueil permettant d'accéder aux différents services.
- **Analyse IA en Temps Réel :** Soumission d'images de lésions cutanées avec calcul instantané du score de confiance.
- **Registre Historique :** Suivi complet des dossiers patients.
- **Interface Moderne :** Thème sombre "Medical Navy & Purple" épuré avec Tailwind CSS.
